# **App Name**: Smart Hospital Dashboard

## Core Features:

- Authentication: Login/Register pages with role selection (Patient/Staff/Admin).
- Main Dashboard Layout: Top navbar, left sidebar navigation with key sections.
- Dashboard Overview: KPI cards, queue trend chart, bed occupancy progress bars.
- OPD Queue Management: Doctor-wise queue displays with patient waiting info.
- Appointment Booking: Patient booking form, token generation, estimated waiting time display.
- Virtual Waiting Room: Table view of patients, queue position, status, countdown timer UI.
- ICU Length-of-Stay (LOS) Prediction: A tool that predicts patient's estimated ICU length-of-stay.

## Style Guidelines:

- Primary color: Soft teal (#64B5CD) to evoke a sense of calm and cleanliness.
- Background color: Light grayish-blue (#E0F7FA) to create a clean and professional backdrop.
- Accent color: Muted blue (#457B9D) for interactive elements and highlights.
- Body and headline font: 'Inter' (sans-serif) for a modern and clean interface.
- Use modern icons from Lucide or Heroicons for a consistent and professional look.
- Card-based layout with good spacing and clear hierarchy for easy navigation.
- Subtle animations for loading states and transitions to enhance user experience.
